# Fazendo _Push_ de Commits para o GitHub

Depois que voce fechou os seus commits, eles podem ser enviados ao repositorio remoto criado ao linkar seu repositorio local ao remoto, e para fazer isso e bem simples:

```bash
  git push
```

Este comando envia os seu commits para a branch principal do seu repositorio remoto(o que e um branch sera explicado em outra secao).

Futuramente sera necessario que voce indique para qual branch os commits devam ser enviados, neste caso voce deve indicar o nome do repositorio remoto e a branch que voce esta trabalhando:

```bash
  git push origin nome-da-branch
```
